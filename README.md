# 背景

之前哈工大的同学找到我，他的老师接的海绵城市系统评价相关项目在最后的提交里需要有一个桌面应用。

尽管对桌面开发没多了解，但想到咱java无所不能，加上顺手谷歌看到javafx能做。

既然能做那就不是问题，帮他这个忙啦～

# 项目介绍

Javafx-havefun 记录了我从javafx一无所知到最后完成交付一个小应用的全过程～

暂且定义为一个初级入门教程吧～
